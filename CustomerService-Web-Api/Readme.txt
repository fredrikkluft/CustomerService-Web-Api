For å kjøre løsningen

1. Unzip prosjektet
2. Gå inn i mappen å trykk på Customer-Web-Api.sln
4. Trykk på ApiController.cs i Controller mappa
3. Løsningen kan nå kjøres 

Ønsker man at dataene skal lagres til neste gang man kjører applikasjonen
må man kommentere ut Database.SetInitializer(new DBinit()); i klassen DBContext.cs på linje 30.

Alle spørsmål som sendes inn kan sees ved å velge Ofte stilte spørsmål eller administrer spørsmål fra menyen og se etter kategorien som det ble sendt inn som.
